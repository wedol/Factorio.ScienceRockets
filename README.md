# Connavar Space and Promethium Science Rocket Silo

A Factorio mod that adds a specialized rocket silo for space science and promethium science.

## Version

**1.0.0**

## Requirements

- **Factorio**: 2.0+
- **Space Age Expansion**: 2.0+

## Description

This mod introduces a custom rocket silo designed specifically for launching space science satellites and supporting promethium science research in Factorio's Space Age expansion.

**Note**: Promethium science functionality is not yet implemented in this version.

## Features

### New Items

- **Connavar Space Science Rocket Silo**: A specialized silo for launching space science rockets
- **Connavar Space Science Rocket Part**: Components used to build the rocket
- **Connavar Space Science Satellite**: A satellite that generates 1000 space science packs when launched

### New Recipes

- **Rocket Silo Recipe**: Crafting recipe for the new silo (requires research)
  - Ingredients: 1000 steel plates, 1000 concrete, 100 pipes, 200 processing units, 200 electric engine units
  - Energy: 30 seconds

- **Rocket Part Recipe**: Crafting recipe for rocket parts (requires research)
  - Category: Rocket building
  - Automated production

## Installation

1. Download the mod folder
2. Place it in your Factorio mods directory:
   - **Windows**: `%APPDATA%\Factorio\mods\`
   - **Linux**: `~/.factorio/mods/`
   - **macOS**: `~/Library/Application Support/factorio/mods/`
3. Enable the mod in Factorio's mod manager
4. Start a new game or add to existing save
